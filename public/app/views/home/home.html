API dos planetas

API (planets)
GET /planets Lista todos os planetas
POST /planets Cria um novo planeta
GET /planets/:id Lista um planeta
PUT /planets/:id Atualiza um planeta
DELETE /planets/:id Deleta um planeta
GET /planets/name/:name Pesquisa um planeta pelo nome
